    Projects

    Project 1

    As technology becomes more advanced and accessible, more and more children are going online, using the Internet, and playing games. To solve this problem, I created an interactive math game in Roblox Studio to enhance the learning experience of elementary school students. Players are able to select what questions they want to answer based on difficulties from 1st to 6th grade. Upon answering 10 questions correctly, they complete the game. If the the player answers a question wrong, they are reset and start from the beginning again. Some inspiration for this project: I have grown up playing games, and one of the first games I started playing was Roblox. Recently, I've been interested in coming up with ideas and creating projects that can benefit other people, and one of them was to create a game in Roblox. Working on this project was the perfect opportunity to use my knowledge from school as well as to learn about the game-making process and how Roblox Studio works.
    More about this project, like how I built it and the challenges I faced, can be found on this website: https://devpost.com/software/roblox-math-game
